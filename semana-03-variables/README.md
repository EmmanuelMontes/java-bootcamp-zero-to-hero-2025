# Semana 3:  Variables en Java / Week 3: Java Variables

## Descripción / Description

Ejercicios y ejemplos enfocados en el uso de variables en Java. 
Exercises and examples focused on the use of variables in Java.

## Objetivos / Objectives

- Comprender qué es una variable y cómo se declara en Java.
- Practicar la asignación, inicialización y modificación de variables.
- Aplicar variables en ejercicios prácticos y pequeños programas.

---
- Understand what a variable is and how it is declared in Java.
- Practice assigning, initializing, and modifying variables.
- Apply variables in practical exercises and small programs.

## Temas incluidos / Topics included

- Declaración y tipos de variables (int, double, boolean, String, etc.).
- Inicialización y asignación de valores.
- Variables locales y globales.
- Convenciones de nombres en Java.
- Ejercicios de manipulación y uso de variables.

---
- Declaration and types of variables (int, double, boolean, String, etc.).
- Initialization and assignment of values.
- Local and global variables.
- Naming conventions in Java.
- Exercises on manipulating and using variables.
